# AgentNexus

AgentNexus is the open-core repository for an Agent-first enterprise gateway.

This repository contains the runnable core, public SDK/API contracts, basic connector runtime, governance foundations, dev deployment profiles, and the open-core admin console.

Before contributing or using AI coding tools, read:

- `AGENTS.md`
- `CONTRIBUTING.md`
- `docs/governance/repository-boundaries.md`
- `docs/governance/ai-coding-rules.md`
